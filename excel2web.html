This project uses comparative analysis in excel. This project has five parts with multiple steps in each part. The goal of this project is to compare data in Excel using comparison tables, pivot tables, and other features. The purpose of this project is to used various excel functions(ie Lookup)to compare the information from the source using tables, graphs, etc. Part one involves copying the medical data file, state list file, and medical data into the workbook.

Part two of this project is all about the set up of the worksheet relating to the comparisons. The steps in this part includes centering and merging the title, adding colored boarders, background, note, and font colors. Part two includes an input section that has includes the wrapping and shape feature. The comparison area is comparing two different states. This area utilizes VLOOUP, IFERROR function, and conditional formatting. The graph in this section shows a pie graph. Protection is also used in this worksheet section.

Part 3 of this project is regarding pivot tables. The first pivot table is the total policy score average based on region. The second pivot table is the Expanded Medicare eligibility based on scores and state. The second pivot table is about the State Expanded Medicare eligibility's score and the sates. The second pivot table uses a slider. The third pivot table is one that I did individually and it displays states and policy score.

Part four of this project is all about filters. The first filter displays total policy that have an average score higher than 5.23. The first filter uses a number filter. The second filter displays states that have correct itemized bills and limit charges to low-income houses. It also filters according to accurate itemized bills. Filter three displayed limit surprise billing that have accurate itemized bills with limit pricing for necessary care along with the filter of scores of 0 only.

Excel formulas, skills, and techniques that are used in this project:

  • Wrapped skills- Wrapping skill was applied to columns labeled "State 1" and "State 2." These columns remained wrapped even after increasing the column width.
  • Conditional formatting- This skill is used to display policy scores that are not equal to 0.
  • Graph- Column graph is used to display the information in the Music Lookup.
  • IFERROR Function- This function is used to display a message when an incorrect state code is entered.
